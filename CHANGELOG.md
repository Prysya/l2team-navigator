# Changelog

Все значимые изменения проекта фиксируются в этом файле.

## [1.5.0] - 2026-07-14

### Добавлено
- Главная страница (роут /) с сеткой-разводкой по разделам
- Компонент MainPage: карточки с иконкой и названием для каждого таба
- Пасхалка: на главной странице ввод `iddqd` открывает модалку «Саша Ролекс Пес»

### Изменено
- TabBar скрыт на главной, показывается только внутри разделов
- Иконка квестов: 📜 → 📋
- 🔥 соски перенесены в колонку названия квеста (вместо отдельной колонки)
- 🗡️ подсветка оружия убрана, оставлена только 🔥

### Исправлено
- Дублирующиеся квесты у Orc Shaman (удалены разделяемые квесты из QUESTS_BY_RACE)
- Мобильная таблица квестов: table-layout fixed, compact padding, column widths

## [1.4.0] - 2026-07-13

### Добавлено
- Таб «Квесты» с категориями: Расовые квесты, Профессии, Цепочка палач храма
- Категория «Профессии»: выбор рассы, типа (1-я/2-я профа) и класса
- Категория «Цепочка палач храма»: 9 квестов 35–38 уровней
- Все 32 вторых профессии (3 in 1 quests) с NPC и координатами
- Расширенные данные: NPC, локации, координаты на карте для 50+ квестов
- Полные пошаговые прохождения для всех первых профессий (спарсены с mw2.wiki)
- Легенда обязательных к выполнению квестов (оружие ⚔️ / соски 🔥)
- Общий компонент WorldMap (draggable + zoom) для карты мира
- Кнопка «📍 Показать на карте» в деталях квеста
- Ссылка на mw2.wiki для квестов без детального прохождения

### Изменено
- Переименованы NPC и локации всех профессий согласно mw2.wiki
- Фильтр класса в профессиях: 1-я показываются базовые классы, 2-я — продвинутые
- Таблица не показывается до выбора класса

### Исправлено
- ActiveTab через regex (react-router basename)
- Анимация слайдера табов через useLayoutEffect
- Некорректные NPC для первых профессий
- Полнота шагов прохождения (раньше было 3 пункта, теперь полный маршрут)

## [1.3.0] - 2026-07-13

### Добавлено
- Path-based routing (react-router-dom) вместо hash
- `BrowserRouter` с basename + `404.html` для GitHub Pages
- Таб «Прокачка» с калькулятором опыта и справочной информацией
- FloatingNav с якорями для навигации по секциям
- Визуальная шкала штрафа (градиент от красного к синему через белый)
- Collapsible таблица «Опыт группы по сравнению с соло»
- Выделенные блоки-предупреждения (warning alerts)
- `.class-tag` стили для тегов классов в таблице книг

### Изменено
- Все табы переведены на TanStack Table (RaidBoss, Spellbook, Recipe)
- Табы на десктопе: иконка сверху, текст снизу
- Бургер-меню: выезжает слева, соцсети внутри
- Полноэкранный бургер на мобилке
- Счётчики (count) перемещены в controls для всех табов
- Таблицы на мобилке без `display: block` — на всю ширину
- Footer: градиентный divider вместо `border-top`
- Floating label: обводка `$color-primary` вместо фона
- Штрафы: переименованы, таблица заменена на шкалу
- Group EXP: убрана колонка HF, скрытая таблица через `<details>`

### Исправлено
- ActiveTab через `useMemo` для react-router
- Слайдер табов через `useLayoutEffect`
- Скругление таблиц (возвращён `overflow: hidden`)
- Класс-теги без отступов
- Footer без отступа над divider
- Таблицы не растягивались на мобилке (удалён `display: block`)
- Скролл тела при открытом бургере

## [1.2.0] - 2026-07-13

### Добавлено
- CHANGELOG.md с историей версий
- GitHub Releases для всех тэгов
- Бургер-меню с левой стороны на мобилке (60% ширины + оверлей)
- Соцсети (YouTube, Telegram, Forum) внутри бургер-меню
- JS-тултипы для бейджей `[S]`, `[SG]`, `[G]` с русским описанием
- Стилизованные подсказки через `data-tooltip` + portal
- Класс `.monster-name` для увеличенного шрифта имени монстра
- Футер с информацией о проекте, прижат к низу экрана
- Кнопка закрытия (✕) в шапке бургер-меню
- Блокировка скролла при открытом бургере

### Изменено
- Header padding перенесён с `body` на `.container` (нет лишнего скролла)
- `min-height: 100vh` только на контейнере, не на `body`
- Floating label — обводка `$color-primary` вместо фона (сбоку до середины, низ открыт)
- Соц-иконки в хедере скрыты на мобилке
- Таблица рецептов — `text-align: center` для всех колонок кроме первой
- `[S]`, `[SG]`, `[G]` — каждый тип в отдельном `<span>` с подсказкой
- Убрано подчёркивание у всех `a` при ховере (глобально)
- Ссылка L2team в футере без подчёркивания
- Ячейки шанс дропа/спойла — `font-size: 14px`

### Исправлено
- Футор обрезался при коротком контенте (flexbox + margin-top: auto)
- Тултипы `[S]` не работали из-за `overflow: hidden` на таблице
- Двойное экранирование `&#39;` в именах монстров
- Скролл не блокировался при открытом бургере
